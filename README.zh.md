# 剧本杀店务管理系统（Flask + MySQL + Vue3）

<div align="right">
  <a href="./README.en.md"><img alt="English" src="https://img.shields.io/badge/English-View-2ea44f"></a>
</div>

本项目用于数据库课程作业/展示：用一个可运行的前后端系统，演示 **认证鉴权、角色分域、锁位→下单→支付链路、事务、触发器、视图/存储过程/函数、报表统计** 等能力。

## 目录结构

- 后端：`app.py`、`models/`、`database.py`、`security_utils.py`
- 前端（Vue）：`frontend-vue/`
- SQL 脚本：
  - 结构迁移：`database/migrations/`
  - 演示增强/演示数据：`database/demo/`
- 封面图源文件（原图）：`images/`
- 项目文档：`docs/`
- 设计图（CDM/E-R/PDM/数据流图）：`docs/design/`

## 快速开始

- 按文档执行：`docs/deploy.md`
