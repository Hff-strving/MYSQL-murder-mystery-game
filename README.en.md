This is a database design & analysis course project. It provides a runnable full-stack system built with **Flask + MySQL + Vue3**, showcasing **authentication/authorization, role-based access, seat locking → order → payment flow, transactions, triggers, views/stored procedures/functions, and reporting**.

## Project Structure

- Backend: `app.py`, `models/`, `database.py`, `security_utils.py`
- Frontend (Vue): `frontend-vue/`
- SQL scripts:
  - Schema migrations: `database/migrations/`
  - Demo enhancements / demo data: `database/demo/`
- Cover image sources: `images/`
- Docs: `docs/`
- Design diagrams (CDM/E-R/PDM/dataflow): `docs/design/`
